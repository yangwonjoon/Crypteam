# 딥러닝을 활용한 가상화폐 자동매매 플랫폼
### Automatic trading System for Cryptocurrency using deep learning
<br>
<p><b>
많은 사람들은 주식 투자를 통하여 자본을 늘리고 싶어한다. 하지만, 주식 투자를 위해서는 많은 시간과 공부가 필요하다. 또한, 인간의 감정과 거짓된 뉴스 등으로 다양한 변수에 의해 혼란을 겪고 잘못된 판단을 할 수 있다. 

이와 같은 변수를 배제하기 위해 머신러닝을 통해 상승과 하락을 예측하는 모델을 만들고, 백테스팅을 통해 모델을 평가하며 해당 모델을 가지고 모의투자와 실제 가상화폐를 매수, 매도할 수 있는 서비스를 제공한다.
</b></p>
<br><br>
